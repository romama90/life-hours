Your data syncs to the cloud so it follows you everywhere.
Takes ~3 minutes to set up once.

Step 1 — Create free Supabase account

Go to supabase.com → New project → any name.
Then go to Settings → API and copy your Project URL and anon public key.

Step 2 — Run this SQL

In Supabase, open the SQL Editor and run:

create table hours ( id uuid default gen_random_uuid() primary key, date text not null, hour integer not null, category text not null, auto_filled boolean default false, updated_at timestamptz default now(), unique(date, hour) ); alter table hours enable row level security; create policy "allow all" on hours for all using (true) with check (true);

Step 3 — Paste your credentials

Credentials saved only on your device.
